Vue是什么：

Vue.js是一门 MVVM 框架，框架的思路是数据映射视图；是数据驱动的，有别于我们过往的 DOM 操作开发思路

MVC 和 MVVM

MVC 和 MVVM 都是设计模式，是框架的开发理念

  1. MVC：M model 是数据层 V view 视图层 C controller 控制器层，控制器是连接 model 和视图的; 一般是单向数据流，通过 Controller 把数据绑定到 view 中
  2. MVVM：M model 数据层 V view 视图层 VM view-model 视图模型层 通过视图模型把数据层和视图层联系起来，可以单向数据绑定

此外，Vue 是响应式的框架；所谓响应式不是css 的响应式，而是通过修改数据，视图自动发生变化。当视图发生变化时，数据也会随着改变；这就是双向数据绑定;

{1 && 2}
