指令是什么

在Vue 中，以 v- 开头的行内属性; Vue 赋予了这些属性一些特殊功能

常用的指令
  1. v-model 把表单元素的 value 和 data 中某个属性绑定在一起
  2. v-text 把数据绑定到 DOM 元素中，且具有响应式，会把之前的覆盖掉，v-text 不识别标签
  3. v-html 可以识别标签
  4. v-if 当属性值是 true ，显示元素，如果为 false 当前元素不显示，但是会显示 v-else 中的内容
  5. v-show 当属性值是true ，显示元素，为false 隐藏元素
  6. v-if是控制的 DOM 元素，如果为 false 页面中没有这个元素；而 v-show 是设置的元素的 display
  7. v-bind 用于绑定动态属性，并且可以简写成一个：,v-bind 后，这个属性就可以使用 data 中该属性的值
